Atalhos gerais

Vamos ver alguns atalhos legais no vs code para facilitar a vida do programador

Alt+z: Quebra automatica de linha

Alt+setas cima/baixo: Move a seleção para cima/baixo no arquivo. se nada estiver selecionado, movimenta a linha em que o cursor está

shift+alt+setas cima/baixo: Move a seleção para cima/baixo no arquivo. se nada estiver selecionado, copia a linha em que o cursor está
